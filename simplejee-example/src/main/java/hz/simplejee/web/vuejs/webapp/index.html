VueJS首页

Vue.js是用于构建交互式Web界面的JS类库，提供了MVVM数据绑定和一个可组合的组件系统。Vue.JS集中在MVVM模式上的视图模型层，并通过双向数据绑定连接视图和模型，实际的DOM操作和输出格式被抽象成指令（Directives）和过滤器（Filters）。

Vue.js 的 API 是参考了 AngularJS、KnockoutJS、Ractive.js、Rivets.js。

核心概念

  1. ViewModel

    一个对象，通过模型和视图。在Vue.JS中，ViewModel是实例化的Vue的构造器或者其子类

    var vm = new Vue({/* options */})
  2. View

    用户看到的实际HTML DOM

    vm.$el
  3. Model

    这是一个略微修改的JavaScript对象

    vm.$data
  4. Directives

    私有的HTML属性告诉Vue.js做一些关于DOM的处理

    <div v-text="message"></div>
  5. Mustache Bindings

    可以使用mustache-style绑定文本和属性，它们会被翻译成v-text,v-attr指令

    <div id="id-{{id}}">Hello {{name}}!<div>

    但是当设置一个image的src属性时，会触发http请求，所以当模板首次加载的时候会出现404，此时用v-attr比较好
